# Analysis

Run the analysis tool from the project root:

```bash
python3 Analysis/analyze.py
```

It writes human-readable reports to:

```bash
Analysis/output/
```

Main outputs:

- `README.txt`
- `emeralds_report.txt`
- `tomatoes_report.txt`
- `bot_comparison.txt`
- `matching_report.txt`
- `results_report.txt`
